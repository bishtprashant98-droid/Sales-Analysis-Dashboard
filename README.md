# Sales Analysis Dashboard | Power BI Project

## Project Overview

This project is an interactive Sales Analysis Dashboard created using Power BI.
It helps track sales performance, profit, customers, and category-wise trends.
The dashboard supports filtering by year, month, region, and category.

## Business Problem

A retail company wants to analyze:

- Total Sales and Profit performance
- Monthly sales trends
- Top customers and profitable categories
- Region-wise sales distribution

This dashboard provides key insights for better decision-making.

## Dataset Details

The dataset contains sales order transactions with the following fields:

- Order ID
- Order Date
- Customer Name
- Category and Sub-category
- Sales Amount
- Quantity
- Discount
- Profit
- Region

## Tools & Technologies Used

- Microsoft Excel (Data Cleaning)
- Power BI Desktop (Dashboard + DAX Measures)
- DAX (Calculated Measures)
- Power Query (Data Transformation)

## Key KPIs Created (DAX Measures)

- Total Sales
- Net Sales
- Total Profit
- Profit Margin %
- Average Profit
- Total Orders
- Total Customers
- Average Delivery Time
 
-Example:
-Profit Margin % =
DIVIDE([Total Profit], [Total Sales], 0)

## Insights & Findings

- Technology category generated the highest sales.
- West region contributed maximum profit.
- Sales peak was observed in March.
- Top 10 customers contribute significant revenue share.
- Discount impacts profit margin in some categories.

## Conclusion

This dashboard provides a complete overview of sales performance and helps stakeholders
monitor profit trends, customer contribution, and regional performance.
